1.font-size : 기본 (16px-1em-100%-12pt-1rem) 15px많이사용
2.font-weight : 글꼴두께 (100~900 / bold-700 / bolder / lighter / normal-400-기본)
3.line-height : 줄간격 (px / em-자주사용 / % / rem)
4.font-family : 글꼴지정 (font name)
5.font-style : 문자스타일-기울림체 (normal / italic / oblipue)
6.color : 글꼴색 (#000000 / rgb(0,0,0) / rgna(0,0,0,0.5)) a는 투명도
7.text-decoration : 텍스트 줄표시-제거 (none / umderline / overline 윗줄 / line-through 취소선)
8.text-transform : 텍스트를 대문자 및 소문자로 변환 (none / capitalize 첫글자만 대문자 / uppercase / lowercase )
9.text-align : 문자 정렬 방법을 지정 (center / left / right / justify 양쪽정렬)
10.text-shadow : 텍스트 그림자 효과 ([x-축거리][y-축거리][퍼짐 거리 정도][색상] / none)

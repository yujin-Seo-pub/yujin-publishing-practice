- 인라인(inline) 요소

1. 한줄에 여러개 배치
2. 기본 너비값은 컨텐츠의 너비값
3. 크기값을 가질 수 없음 (width, height X)
4. 상하 마진은 가질 수 없음 (좌우마진은 가능)

span a small big em u ins i s del br q b strong mark sub sup label map cite abbr time

- 블록(block) 요소

1. 한줄에 한개만 배치
2. 기본 너비값은 100% 정해짐
3. 크기값을 가질 수 있음 (width, height O)
4. 상하 마진은 가질 수 있음

div p table blockquote ul ol li dl td dd table tr td th form hr h1~h6 header nav footer main section article aside figure figcaption summary details address

- 인라인블록(inline-block) 요소

1. 한줄에 여러개 배치
2. 기본 너비값은 컨텐츠의 너비값
3. 크기값을 가질 수 있음 (width, height O)
4. 상하 마진은 가질 수 있음
-> 인라인블록 요소는 한줄에 여러개 배치해야하고 마진줘야하는 경우사용

img input 태그들 botton fontawesome

- padding은 인라인요소나 블록요소나 모두 상하좌우 마진 가질수 있음

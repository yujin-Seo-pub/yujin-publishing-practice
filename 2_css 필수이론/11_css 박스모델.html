< box-sizing: border-box > pdf 44p 그림참고

보더 크기는 padding과 border size를 주게되면 처음 정했던 width, height 값에 더해져서 커지게 되는데

box-sizing: border-box를 사용하면 padding과 border size를 주게되도 처음 정했던 width, height 값안에 포함되게 된다.

.box{
width: 600px;
height: 50px;
padding: 20px;
border: 10px solid red;
}

width 600인데 양쪽 padding (20+20) 과 양쪽 border size (10+10) 총 660 이 되버린다

그러나 box-sizing: border-box 사용하면 width 값 600 안에 padding 40과 border 20 포함되서 총 600 으로 보여진다.

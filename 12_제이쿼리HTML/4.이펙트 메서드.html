<제이쿼리 메서드>

★ 메서드 (이펙트)

.show : 선택요소 보이기

.hide : 선택요소 감추기

.toggle : 선택요소 보이기/감추기

.slideDown : 선택요소 서서히 보이기

.slideUp : 선택요소 서서히 감추기

.slideToggle : 선택요소 서서히 보이기/감추기

.fadeIn : 선택요소 페이드로 나타나게 하는 효과

.fadeOut : 선택요소 페이드로 사라지게 하는 효과

.fadeToggle : 선택요소 페이드로 나타나고 사라지게 하는 효과

.stop : 애니메이션 멈추는 역할

이펙트 메서드의 지속시간은 fadeIn('') 괄호 안 따옴표 안에slow fast 적어주기 또는 괄호안에 (500) (3000) 숫자만 넣어주기

★ 메서드 (클래스 요소)

.addClass : 선택요소에 클래스 넣기

.removeClass : 선택요소에 클래스 뺴기

.toggleClass : 선택요소에 클래스 넣기/뺴기

클래스 제어 메서드의 클래스명은 괄호안에 .쓰지말고 그냥 써야됨 addClass('ative')O ('.active')X

★ 메서드 (요소탐색)

.children

.parent

.siblings : 선택요소의 형제요소

.find : 선택요소의 자손요소

.next : 선택요소 바로 다음 요소

.prev : 선택요소 바로 전 요소

요소탐색 메서드의 클래스명은 괄호안에 .써야됨 children('.ative')O ('active')X
